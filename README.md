# @amlplugins/mistral-fine-tuning

Native `@mistralai/mistralai` SDK re-exported under the `@amlplugins` namespace.

> Mistral Fine-Tuning — supervised fine-tuning jobs, classifier training, hyperparameter config.

## Install

```bash
npm install @amlplugins/mistral-fine-tuning
```

## Use

```ts
import { /* named exports from @mistralai/mistralai */ } from "@amlplugins/mistral-fine-tuning";
// or
import sdk from "@amlplugins/mistral-fine-tuning";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
